# Job Market Analysis Dashboard

## Objective
Analyze job market data to identify in-demand skills, salary trends, and hiring patterns for data-related roles.

## Dataset
- 2,200+ job postings
- Source: Glassdoor job listings

## Tools Used
- Python (Pandas, Matplotlib)
- Excel (Pivot Tables, Charts)
- Jupyter Notebook

## Key Analysis
- Top hiring locations
- Highest paying job titles
- Top hiring companies
- Skill demand from job descriptions

## Dashboard
Interactive Excel dashboard built using pivot tables and charts to visualize insights.

## Key Insights
- New York, Chicago, and San Francisco have the highest job openings
- Python and SQL are the most demanded skills
- Data Analyst roles dominate the market
- Senior roles have higher average salary
